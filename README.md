Auto-XML
========

An automatic XML generator for PGM

## Information

* Written in VB, as a console application
* Simply written, should be easy to understand, as intended
* Read the contributing.md document.

## Current features

* Map proto auto-adding
* Map name
* Map version
* Objective
* Author (single only)
* Teams (2 only)
* Kits
